Local files / Incognito

To enable playback of local media (File > Open File) or Incognito mode, grant additional permissions:

  • Navigate to chrome://extensions
  • Find "Video Speed Controller" and click Details
  • Enable "Allow access to file URLs" and/or "Allow in Incognito"
Disable on a site

Add the site's domain to the blacklist in Other Settings (e.g. example.com). Regex is supported (e.g. /\.edu$/i).

Per-site speed

This fork automatically remembers speed per website when "Remember playback speed" is enabled. Speeds are stored per video source URL and restored on revisit.

Shortcut conflicts

If a site captures one of your shortcut keys, try Shift+<key> — the extension listens for both cases. You can also remap keys in Key Bindings above.

Controller not showing

The extension only works with HTML5 video/audio. If you don't see controls, the site may be using a non-standard player. Try reloading the page. For very small or zero-size videos, the controller is hidden automatically.

Speed resets on navigation

Single-page apps (YouTube, Netflix) may reset speed when navigating between videos. With "Remember playback speed" enabled, the extension will re-apply your saved speed.

Audio support

Enable "Also control audio elements" in Other Settings to show speed controls on <audio> elements (podcasts, music players, etc.).
